# Breakers of E

A Magic: The Gathering collection manager, deck builder, and tabletop simulator for Windows.

---

## Download

👉 **[Download the latest installer](../../releases/latest)**

## Features

### Collection Management
- Browse all ~98,000 Magic cards from Scryfall
- Track your collection with quantities, foil status, condition, language, storage location, and personal prices
- Import from MTG Studio, Moxfield, TCGPlayer, Deckbox, and Dragon Shield
- Export to all major formats

### Deck Builder
- Build Standard and Commander decks
- Pool → Deck or Collection → Deck mode
- Full deck statistics: mana curve, color pie, legality, card value, and more
- Foil card support, commander designation, sideboard

### Trade Binder & Want List
- Visual card binder — just like a real physical binder
- Track cards you want to trade away (Have List) and cards you're looking for (Want List)

### Tabletop Simulator
- Play Magic against yourself or a friend
- Full Commander support: command zone, commander tax, commander damage tracking
- Life totals, poison counters, phases, the stack, tokens, and more
- Modern mechanics: Monarch, Initiative, Energy, The Ring
- Save and restore game state

### Tools
- Keyword Search — find cards by ability keywords with color and legality filters
- Keyword Dictionary — full MTG rules glossary (~200 keywords)
- Advanced Filter — multi-condition expression builder
- Card images downloaded from Scryfall and cached locally

## Getting Started

1. **Install** — run `BreakersOfE_Setup_v1.0.0.exe` and follow the wizard
2. **Download the card database** — check the option during install, or go to **Tools → Update Card Database** after launch
3. **Add your collection** — switch to **Pool → Collection** mode and start adding cards, or import from another app via **File → Import**
4. **Build a deck** — switch to **Pool → Deck** or **Collection → Deck** and create a new deck with **File → New Deck**

---

## System Requirements

- Windows 10 or later (x86 or x64)
- Internet connection required for first-time card database download
- ~500 MB disk space for card images (downloaded on demand)
